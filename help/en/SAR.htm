Command: SAR- Shift arithmetic right
Arguments: OP: register or memory location; QUANTITY: immediate
Usage: SAR OP, QUANTITY
Effects: Shifts all bits of OP right QUANTITY times and fills up with the most significant bit of the input.
Flags to be set: OF, SF, ZF, PF, CF
Misc: Carry flag contains bit that was shifted out last.
Overflow flag is only affected with 1-bit shifts and is set to 0. If QUANTITY isn't 1, the overflow flag's state is undefined!
Corresponds to a signed division by 2 QUANTITY times while rounding the quotient toward negative infinity.
