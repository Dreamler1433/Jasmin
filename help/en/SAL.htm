Command: SAL- Shift arithmetic left
Arguments: OP: register or memory location; QUANTITY: immediate
Usage: SAL OP, QUANTITY
Effects: Shifts all bits of OP left QUANTITY times; fills up with 0.
Flags to be set: OF, SF, ZF, PF, CF
Misc: Carry flag contains bit that was shifted out last.
Overflow flag is only affected with 1-bit shifts and is set to 0 if the most-significant bit of the result is the same as the carry flag; otherwise it's set to 1. If QUANTITY isn't 1, the overflow flag's state is undefined!
Corresponds to multiply by 2 QUANTITY times.
